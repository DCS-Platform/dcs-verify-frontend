Steps 1–5 run automatically on every build. A 60-agent build produces 60 linked receipts in one session chain — the more the pipeline does, the longer the proof. Algorithm Ed25519 · keys rotate yearly (2026 → 2027), and old receipts stay verifiable against the key that signed them.
